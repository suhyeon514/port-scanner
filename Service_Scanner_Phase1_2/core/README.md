# README

## 프로젝트 개요
이 프로젝트는 Nmap을 활용한 네트워크 스캔 및 결과 분석을 자동화하는 도구입니다. 주요 기능은 Nmap 스캔 결과를 파싱하고, 1차 및 2차 스캔을 통해 포트 상태와 서비스 정보를 수집하며, 최종적으로 JSON 형식의 리포트를 생성하는 것입니다.

---

## 파일 설명

### 1. `nmap_parser.py`
- **역할**: Nmap의 XML 출력 결과를 JSON 형식으로 변환하는 파서입니다.
- **주요 기능**:
  - XML 파일 또는 스트림을 읽어 JSON으로 변환.
  - Nmap 스캔 메타데이터, 호스트 정보, 포트 상태 및 서비스 정보를 파싱.
- **사용 예시**:
  - CLI를 통해 XML 파일을 JSON으로 변환.
  - 다른 코드에서 모듈로 임포트하여 사용 가능.
- **상호 관계**:
  - `nmap_runner.py`에서 Nmap 스캔 결과(XML)를 파싱하는 데 사용됩니다.

---

### 2. `nmap_report.py`
- **역할**: 1차 및 2차 스캔 데이터를 병합하여 최종 리포트를 생성합니다.
- **주요 기능**:
  - 1차 스캔(phase1)과 2차 스캔(phase2) 데이터를 통합.
  - 각 호스트별로 포트 상태와 서비스 정보를 정리.
  - 최종 리포트를 JSON 형식으로 반환.
- **상호 관계**:
  - `nmap_runner.py`에서 1차 및 2차 스캔 데이터를 받아 최종 리포트를 생성합니다.

---

### 3. `nmap_runner.py`
- **역할**: Nmap 스캔을 실행하고 결과를 처리합니다.
- **주요 기능**:
  - 1차 스캔(phase1): 지정된 프로파일과 타겟에 따라 Nmap 실행.
  - 2차 스캔(phase2): 1차 스캔 결과를 기반으로 추가 스캔 수행.
  - 스캔 결과(XML)를 JSON으로 변환하고 저장.
- **상호 관계**:
  - `nmap_parser.py`를 사용하여 Nmap XML 결과를 파싱.
  - `nmap_report.py`를 호출하여 최종 리포트를 생성.

---

### 4. `orchestrator.py`
- **역할**: Nmap 스캔을 실행하기 위한 CLI 인터페이스를 제공합니다.
- **주요 기능**:
  - 사용자 입력을 받아 Nmap 스캔 실행.
  - 프로파일 및 타겟 정보를 로드하여 Nmap 명령어를 구성.
- **상호 관계**:
  - `nmap_runner.py`를 호출하여 스캔을 실행합니다.

---

## 파일 간 상호 관계
1. **`orchestrator.py`**는 사용자 입력을 받아 **`nmap_runner.py`**를 호출하여 스캔을 실행합니다.
2. **`nmap_runner.py`**는 Nmap 스캔을 실행하고 결과(XML)를 **`nmap_parser.py`**를 통해 JSON으로 변환합니다.
3. **`nmap_runner.py`**는 1차 및 2차 스캔 데이터를 **`nmap_report.py`**로 전달하여 최종 리포트를 생성합니다.
